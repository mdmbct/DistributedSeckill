# 分布式秒杀、抽奖、抢红包

​                                                                                                                                                                                         --学习、开发中

## 简介

一个通用的、支持分布式的秒杀、抽奖、抢红包模块。

### 模块

seckill-core：核心功能实现，在项目中必须引入。

seckill-starter：springboot-starter，快速配置，快速使用。

seckill-app：示例微服务，可部署集群。
